# Código de conducta

#### __Objetivo__

1. Establecer un marco de referencia para las decisiones y acciones de todas
las personas usuarias y administradoras.
2. Establecer el conjunto de principios, valores y reglas para definir las
expectativas de comportamiento.
3. Promover un entorno de trabajo y colaboración confiable y respetuoso.
4. Promover la igualdad, integridad y privacidad de las personas usuarias.
5. Promover la seguridad y sustentabilidad del sistema.

#### __Contrato__

Las personas usuarias y administradoras de este servidor público y soberano
(a partir de aquí ¨sobnix¨) aceptan cumplir con las disposiciones de las
siguientes políticas:

1. [Política de igualdad](https://sobnix.ar/politica_igualdad.html).
2. [Política de integridad](https://sobnix.ar/politica_integridad.html).
3. [Política de privacidad](https://sobnix.ar/politica_privacidad.html).
4. [Política de seguridad](https://sobnix.ar/politica_seguridad.html).
5. [Política de sustentabilidad](https://sobnix.ar/politica_sustentabilidad.html).
